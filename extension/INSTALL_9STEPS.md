# 9步工作流扩展安装指南

## 📦 安装扩展

### 方法1：通过 VSIX 文件安装（推荐）

1. **打开 Cursor/VS Code**
2. **打开扩展面板**：按 `Ctrl+Shift+X` 或点击左侧扩展图标
3. **安装 VSIX**：
   - 点击扩展面板右上角的 `...` 菜单
   - 选择 `从 VSIX 安装...`
   - 选择文件：`extension/trquant-extension-9steps.vsix`
4. **重新加载窗口**：安装完成后，按 `Ctrl+Shift+P`，输入 `Developer: Reload Window`

### 方法2：通过命令行安装

```bash
cd /home/taotao/dev/QuantTest/TRQuant/extension
code --install-extension trquant-extension-9steps.vsix
```

## 🚀 使用9步工作流

### 方式1：通过侧边栏

1. **打开侧边栏**：点击左侧活动栏的 **TRQuant** 图标（📊）
2. **找到工作流视图**：在侧边栏中找到 **🚀 工作流** 视图
3. **打开9步工作流面板**：
   - 点击视图标题栏的 **工作流图标** 按钮，或
   - 点击树视图中的 **🚀 9步工作流面板** 项

### 方式2：通过命令面板

1. **打开命令面板**：按 `Ctrl+Shift+P`
2. **输入命令**：输入 `TRQuant: 9步工作流` 或 `openWorkflowV2`
3. **选择命令**：选择 `TRQuant: 9步工作流`

### 方式3：通过侧边栏工作流树

在侧边栏的 **🚀 工作流** 视图中，你可以看到：

- **🚀 9步工作流面板** - 点击打开完整工作流界面
- **📡 1. 信息获取** - 数据源配置和数据更新
- **📈 2. 市场趋势** - 市场状态和趋势分析
- **🔥 3. 投资主线** - 识别投资主线
- **📦 4. 候选池构建** - 构建股票候选池
- **📊 5. 因子构建** - 因子推荐和配置
- **🛠️ 6. 策略生成** - 策略代码生成（点击打开策略生成器）
- **🔄 7. 回测验证** - 回测执行（点击打开回测面板）
- **⚡ 8. 策略优化** - 参数优化（点击打开优化面板）
- **📄 9. 报告生成** - 报告生成（点击打开报告面板）

## 🔧 故障排除

### 问题1：看不到工作流视图

**解决方案**：
1. 确认扩展已安装：`Ctrl+Shift+X` → 搜索 "TRQuant"
2. 重新加载窗口：`Ctrl+Shift+P` → `Developer: Reload Window`
3. 检查侧边栏：点击左侧活动栏的 TRQuant 图标

### 问题2：点击工作流没有反应

**解决方案**：
1. 打开开发者工具：`Ctrl+Shift+P` → `Developer: Toggle Developer Tools`
2. 查看控制台错误信息
3. 检查 MCP 服务器是否运行

### 问题3：MCP 工具调用失败

**解决方案**：
1. 确认 Python 后端已启动
2. 检查 MCP 服务器配置
3. 查看扩展日志：`Ctrl+Shift+P` → `TRQuant: 显示日志`
