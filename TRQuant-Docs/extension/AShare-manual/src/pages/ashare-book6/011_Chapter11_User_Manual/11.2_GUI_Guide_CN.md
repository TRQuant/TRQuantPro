## 🎯 学习目标

通过本节学习，您将能够：

- **理解界面布局**：掌握主窗口、侧边栏、内容区的布局结构
- **使用功能模块**：掌握各个功能模块的操作方法
- **执行操作流程**：掌握模块切换、数据操作、结果查看等操作流程
- **管理系统**：掌握数据管理、系统设置、运行日志的使用方法

## 📚 核心概念

### 界面结构

- **主窗口**：整体框架，包含侧边栏和内容区
- **侧边栏**：导航栏，包含所有功能模块入口
- **内容区**：功能面板显示区域，使用StackedWidget管理

### 功能模块

- **工作台**：系统概览、快速入口、工作流集成
- **量化工作流**：8步骤工作流（信息获取→市场趋势→投资主线→候选池→因子构建→策略生成→回测验证→实盘交易）
- **系统管理**：数据管理、系统设置、运行日志

### 侧边栏导航

侧边栏包含以下导航项：

#### 量化工作流

- **🏠 工作台**：系统概览、快速入口、工作流集成
- **📡 信息获取**：数据源管理、知识库查询、资讯收集
- **📈 市场趋势**：市场趋势识别（短/中/长期）
- **🔥 投资主线**：五维量化→综合评分→主线识别
- **📦 候选池**：股票池+ETF池构建
- **📊 因子构建**：因子库+计算+组合
- **🛠️ 策略开发**：策略生成、策略编辑、策略测试
- **🔄 回测验证**：策略回测、回测分析
- **🚀 实盘交易**：实盘交易、订单管理、风险控制

#### 系统管理

- **📁 数据管理**：文件管理系统
- **⚙️ 系统设置**：系统配置、数据源配置
- **📋 运行日志**：查看系统运行日志

### 内容区域

内容区域使用`QStackedWidget`管理多个功能面板，支持延迟加载：

- **延迟加载**：只加载当前显示的面板，提升启动速度
- **面板切换**：点击侧边栏按钮切换面板
- **状态保持**：面板状态在切换时保持

<h2 id="section-11-2-2">🧩 11.2.2 功能模块</h2>

功能模块包括工作台、量化工作流各步骤、系统管理等。

### 工作台（首页）

**功能**：

- 系统概览卡片
- 快速入口工具卡片
- 集成工作流程面板
- 最近操作记录

**操作**：

1. 查看系统状态卡片（数据源连接、市场分析结果）
2. 点击工具卡片快速进入功能模块
3. 使用集成工作流程面板执行完整工作流

### 信息获取

**功能**：

- 数据源管理（JQData、AKShare、Wind）
- 数据源连接状态
- 数据质量检查
- 知识库查询

**操作**：

1. 点击"添加数据源"配置数据源
2. 点击"测试连接"验证数据源
3. 查看数据源连接状态
4. 执行数据质量检查

### 市场趋势

**功能**：

- 市场趋势分析（短/中/长期）
- 市场状态判断（risk_on/risk_off/neutral）
- 指数趋势分析
- 风格轮动分析

**操作**：

1. 点击"开始分析"执行市场分析
2. 查看市场状态卡片
3. 查看指数趋势图表
4. 查看风格轮动结果

### 投资主线

**功能**：

- 投资主线识别
- 主线评分
- 主线筛选
- 主线详情查看

**操作**：

1. 点击"识别主线"执行主线识别
2. 查看主线列表和评分
3. 点击主线查看详情
4. 基于主线生成策略或执行回测

### 候选池

**功能**：

- 候选股票池构建
- 股票筛选
- 股票评分
- 股票池管理

**操作**：

1. 选择主线或筛选条件
2. 点击"构建候选池"执行构建
3. 查看候选股票列表
4. 查看股票评分和排名

### 因子构建

**功能**：

- 因子库浏览
- 因子计算
- 因子组合
- 因子优化

**操作**：

1. 浏览因子库，选择因子
2. 点击"计算因子"执行计算
3. 查看因子计算结果
4. 组合多个因子，设置权重

### 策略开发

**功能**：

- 策略生成器
- 策略编辑器
- 策略测试
- 策略管理

**操作**：

1. 选择策略风格（多因子/动量成长/价值/市场中性）
2. 选择因子列表
3. 设置风险参数
4. 选择目标平台（PTrade/QMT）
5. 点击"生成策略"生成策略代码
6. 编辑策略代码（如需要）
7. 保存策略

### 回测验证

**功能**：

- 策略回测
- 回测结果分析
- 回测报告导出
- 回测历史管理

**操作**：

1. 选择策略文件
2. 设置回测参数（开始日期、结束日期、初始资金、手续费率）
3. 点击"开始回测"执行回测
4. 查看回测结果（收益曲线、交易记录、风控报告）
5. 导出回测报告

### 实盘交易

**功能**：

- 券商连接（PTrade/QMT）
- 订单管理
- 持仓管理
- 风险控制

**操作**：

1. 配置券商连接
2. 连接券商平台
3. 查看持仓和订单
4. 执行交易操作
5. 监控风险指标

<h2 id="section-11-2-3">🔄 11.2.3 操作流程</h2>

操作流程包括模块切换、数据操作、结果查看等。

### 模块切换

#### 方法1：侧边栏导航

1. 点击侧边栏导航按钮
2. 内容区自动切换到对应面板
3. 面板延迟加载（首次访问时加载）

#### 方法2：快速入口

1. 在工作台点击工具卡片
2. 自动跳转到对应功能模块

### 数据操作

#### 数据输入

- **表单输入**：在输入框中输入数据
- **文件选择**：使用文件选择对话框选择文件
- **下拉选择**：从下拉列表中选择选项

#### 数据提交

- **按钮提交**：点击"确定"、"保存"等按钮提交数据
- **自动保存**：某些操作自动保存（如策略编辑）

### 结果查看

#### 表格查看

- 在表格中查看数据列表
- 支持排序、筛选、搜索
- 支持导出为CSV/Excel

#### 图表查看

- 查看各种图表（折线图、柱状图、饼图等）
- 支持图表交互（缩放、平移）
- 支持图表导出

#### 报告查看

- 查看HTML报告
- 查看PDF报告
- 导出报告文件

### 文件管理

#### 文件浏览

1. 打开"数据管理"面板
2. 浏览文件树
3. 查看文件详情

#### 文件操作

- **打开文件**：双击文件或右键选择"打开"
- **编辑文件**：右键选择"编辑"或"在Cursor中打开"
- **删除文件**：右键选择"删除"
- **运行回测**：右键选择"运行回测"（策略文件）

<h2 id="section-11-2-4">⚙️ 11.2.4 系统管理</h2>

系统管理包括数据管理、系统设置、运行日志等。

### 数据管理

**功能**：

- 文件浏览（策略、回测结果、数据文件）
- 文件操作（打开、编辑、删除、运行回测）
- 文件搜索
- 文件分类管理

**操作**：

1. 打开"数据管理"面板
2. 浏览文件树
3. 执行文件操作（打开、编辑、删除等）

### 系统设置

**功能**：

- 系统启动/停止
- 数据源配置
- 系统检查（配置、模块、网络）
- 环境变量配置

**操作**：

1. 打开"系统设置"面板
2. 点击"系统检查"验证环境
3. 配置数据源
4. 点击"启动系统"启动系统服务

### 运行日志

**功能**：

- 查看系统运行日志
- 日志级别过滤
- 日志搜索
- 日志导出

**操作**：

1. 打开"运行日志"面板
2. 查看实时日志
3. 使用搜索功能查找特定日志
4. 导出日志文件
